# Fable Prompt Set — how to run

**Every run = 3 parts pasted in order:**
1. `BOUNDED_BIND_CONTEXT.md`  (the technical pack — the fixed prefix)
2. `prompt-00-shared-context.md`  (business/company context — same every time)
3. ONE themed task below

Each themed prompt is self-contained given 1+2 and ends in a decision + concrete
actions (owners, budget, kill/scale metric), not a menu of maybes.

| File | Task |
|---|---|
| prompt-01-business-direction-portfolio.md | Overall direction: big menu of routes → a decided portfolio + 30/60/90 plan + budget |
| prompt-02-bounded-positioning-gtm.md | If Bounded is the bet: positioning, ICP, homepage rewrite, launch plan, 3 demos |
| prompt-03-bounded-enterprise.md | Enterprise Bounded: pursue now/later/not, design-partner plan, must-build list |
| prompt-04-live-edit-internal-tool-builder.md | Live-edit "internal tool builder": product vs feature, governance/forking model, MVP |
| prompt-05-ai-investment-desk.md | Fable-as-investment-desk: system design, honest edge, capital to commit, go/no-go gates |
| prompt-06-ratri-india-gtm.md | Ratri India launch: wedge, Razorpay, INR pricing, distribution, first experiment |
| prompt-07-product-factory.md | Product factory OS + first 3–5 bets (tennis sim, cutscene revival, +new) |

Tip: run 01 first to pick the portfolio, then run the themed prompts for whatever it
tells you to pursue. You can also run several in parallel loops.
