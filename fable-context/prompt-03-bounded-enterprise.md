================================================================================
TASK 3 — BOUNDED ENTERPRISE (a big bet we're weighing)
================================================================================

Using the technical pack + shared business context above, evaluate and design an
**enterprise version of Bounded**. We think this could be a big version of the company.

## Do this

1. **What "enterprise Bounded" actually is.** Translate the core (provable policy,
   invariants, atomic data plane, proof reports) into enterprise value: **compliance
   & audit** (proof reports / counterexamples as compliance artifacts), **self-host /
   VPC / on-prem / BYOC**, SSO/SAML/SCIM, RBAC, SLAs, data residency, support. What of
   this exists today vs must be built (respect the maturity flags — e.g. multi-tenant
   isolation, BACKEND-004 atomicity).
2. **ICP & buyer.** Which enterprises feel the "prove the backend can't go wrong" pain
   most (fintech, healthcare, gaming/anti-cheat, multi-tenant SaaS, agent platforms)?
   Who's the economic buyer vs champion?
3. **Packaging & pricing.** Editions, price points, what gates enterprise vs Pro,
   contract shape (annual, seat vs usage vs platform fee).
4. **Sales motion.** Design-partner program, POC/pilot structure, proof-of-value that
   lands in weeks, and the security/compliance collateral we'd need (SOC2, pen-test —
   we already run honest audits per the pack).
5. **Cost of the bet.** Honestly weigh: a **4-person team** doing enterprise sales +
   support vs shipping velocity. What breaks, what we'd have to hire, and the risk it
   pulls us off everything else.

## End in action, with conviction

- **A decision:** pursue enterprise **now / later / not** — with the reasoning.
- If yes: a **design-partner acquisition plan** (who to target, exact outreach, the
  pilot offer), the **must-build list in priority order** with build cost, and a
  **30/60/90-day** action plan tagged by owner and budget.
- The **first paying enterprise logo** we should realistically chase, and the metric
  that proves the motion works. Decide; don't hedge.
